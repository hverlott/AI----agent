# 知识库用户手册

## 打开入口
- 在多平台管理后台左侧选择“📚 知识库”，进入知识库配置面板

## 功能分区
- 管理：查看、创建、编辑、删除知识条目
- 导入：上传 txt/md/pdf/docx/xlsx 文件，自动或半自动解析为文本条目
- 检索测试：输入关键词或问题，查看 Top-N 命中条目与摘要
- 设置：检查解析依赖（PyPDF2、python-docx、openpyxl）

## 操作指南
- 新建条目：在“管理”页填写标题/分类/标签与正文，点击保存
- 导入文件：上传文件后可预览和编辑解析文本，保存为条目
- 标签管理：在保存/编辑时以逗号分隔输入多个标签
- 分类管理：在保存/编辑时输入一个分类名称

## 与机器人联动
- 当 QA 未命中时，机器人会将知识库 Top-2 命中条目作为参考上下文注入 AI 会话，提高专业回复质量

## 常见问题
- 解析依赖缺失：在“设置”页查看缺失包名称，安装后重试
- 大文件解析：如超长内容会自动截断至 1200 字以内用于上下文注入
