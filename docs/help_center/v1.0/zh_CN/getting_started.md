# 系统登录与快速入门

Version: 1.0
Last Updated: 2026-01-07

## 1. 系统启动与登录

本系统为本地化部署的 AI 客服管理工具，支持通过 Web 界面进行统一管理。

### 1.1 启动管理后台
1. 打开系统根目录 `D:\AI Talk`。
2. 双击运行 `start_admin.bat` (Windows) 或 `start_admin.sh` (Linux/Mac)。
3. 系统将自动打开默认浏览器，访问 `http://localhost:8501`。

![系统启动界面示意图]

### 1.2 身份与租户切换
系统支持多租户与多角色管理，适用于不同业务线的隔离管理。

1. **租户切换 (Tenant)**:
   - 在左侧侧边栏找到 "租户" (Tenant) 下拉框。
   - 选择对应的租户 ID（默认为 `default`）。
   - 切换后，所有配置、账号、日志数据将自动切换至该租户。

2. **角色切换 (Role)**:
   - 当前版本默认使用 **SuperAdmin** (超级管理员) 权限。
   - 系统已移除手动角色切换功能，所有登录用户均拥有完整系统权限，包括配置管理、会话监管与审计功能。

## 2. 界面概览

系统主要分为以下几个功能区域：

- **侧边栏 (Sidebar)**: 全局导航、身份/租户切换、系统信息。
- **顶部状态栏**: 显示当前选中平台及其运行状态。
- **主工作区**: 根据选择的模块显示具体配置与数据。

## 3. 常见问题 (FAQ)

*   **Q: 启动后浏览器未自动打开怎么办？**
    *   A: 请手动复制终端显示的 URL (通常是 `http://localhost:8501`) 到浏览器访问。
*   **Q: 忘记了 Admin 密码？**
    *   A: 当前版本为本地部署，默认无强制登录鉴权。如开启了 Basic Auth，请联系运维重置 `.streamlit/secrets.toml`。
*   **Q: 为什么切换租户后数据是空的？**
    *   A: 新租户默认没有任何配置，需重新进行初始化配置（账号录入、AI 配置等）。
