# 系统设置与权限管理

Version: 1.0
Last Updated: 2026-01-07

## 1. 功能概述
本模块涵盖系统的基础设施配置（环境变量、密钥）、API 网关管理以及安全权限控制。

## 2. 操作步骤

### 2.1 系统配置自动化 (System Config)
侧边栏选择 **平台** -> **系统配置自动化**。

- **环境配置 (.env)**:
    - 查看当前系统环境变量状态。
    - 点击 **一键生成/更新** 可自动根据模板修复缺失配置。
- **敏感信息查看**:
    - 系统默认隐藏 API Key 等敏感信息。
    - 点击 **生成二次验证码** -> 终端查看验证码 -> 输入验证码 -> 点击 **查看明文** 解锁显示。

### 2.2 API 网关管理 (Gateway)
侧边栏选择 **平台** -> **API接口管理中心**。

- **路由管理**: 配置系统对外暴露的 API 路径 (e.g., `/audit`, `/reply`)。
- **流控策略**: 设置接口的限流阈值 (Rate Limit)，防止滥用。
- **鉴权配置**: 启用/禁用接口的 API Key 验证。

### 2.3 权限管理 (RBAC)
当前版本采用 **统一 SuperAdmin 模式**：

- **权限范围**: 所有访问管理后台的用户默认拥有最高权限 (SuperAdmin)。
- **功能访问**: 可无限制访问系统配置、API 网关、业务监管 (Supervisor) 及审计 (Audit) 等所有模块。
- **设计初衷**: 简化本地部署单机模式下的操作流程，提升运维效率。

*注：多用户分级鉴权功能将在后续企业版中提供。*

## 3. 常见问题 (FAQ)

*   **Q: 如何修改 API 端口？**
*   **Q: 忘记了二次验证码？**
    *   A: 二次验证码输出在**运行后台的终端/命令行窗口**中，非浏览器页面上。请切换到后台窗口查看。
