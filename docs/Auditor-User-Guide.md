# 审核员用户手册

## 1. 进入审核配置
1. 打开管理后台（`admin_multi.py`）。
2. 在左侧侧边栏“👤 身份切换”中选择 **Auditor**（审核员）。
3. 系统将切换至“🛡️ 审核员配置中心”。

## 2. 关键词管理
审核员可以配置两类关键词，配置完成后**实时生效**，无需重启服务。

### 2.1 违禁词 (Block)
- **定义**：绝对禁止出现的内容。
- **行为**：一旦用户输入或AI回复中包含此类词汇，系统将**直接拦截**回复，并触发兜底话术。
- **操作**：
  - **添加**：在“违禁词”栏输入词汇，点击“添加”。
  - **删除**：在下拉框中选择要删除的词汇，点击“删除选中”。

### 2.2 敏感词 (Sensitive)
- **定义**：需要关注但未必拦截的内容。
- **行为**：系统会记录日志（Warning级别），并根据策略可能拒绝或仅做标记。
- **操作**：与违禁词操作一致。

## 3. 审核日志
- 点击“审核日志”标签页。
- 可以查看最近的审核拦截记录，包括触发时间、触发关键词、处理结果等。
- 点击“刷新日志”获取最新数据。

## 4. 常见问题
- **生效延迟**：通常为毫秒级，文件保存后立即生效。
- **日志未更新**：请点击“刷新日志”按钮。
