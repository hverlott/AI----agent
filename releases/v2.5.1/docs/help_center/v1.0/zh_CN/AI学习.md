#+#+#+#+## AI 学习中心

Version: 2.5.1
Last Updated: 2026-01-12

## 1. 功能用途

AI 学习中心用于对“对话采集数据”进行筛选、清洗与标注，形成可用于回归验证或后续学习的训练集。

数据隔离规则：按“租户”隔离，切换租户后仅展示当前租户的数据。

## 2. 入口与权限

- 面包屑：左侧菜单「🤖 AI 配置中心」>「🧪 AI学习中心」

| 角色 | 是否可用 | 租户范围 |
| :--- | :--- | :--- |
| 超级管理员（SuperAdmin） | 可用 | 可切换租户后操作不同租户数据 |
| 业务管理员（BusinessAdmin） | 可用 | 仅可操作绑定租户的数据 |

## 3. 数据浏览与清洗

入口：进入 AI 学习中心后，选择「📄 数据浏览与清洗」标签页。

### 3.1 筛选条件

1. 选择「平台」：`telegram` / `whatsapp` / `all`
2. 选择「方向」：`inbound` / `outbound` / `all`
3. 选择「可学习」：`1`（可学习）/ `0`（不可学习）/ `all`
4. 选择「垃圾」：`1`（垃圾）/ `0`（非垃圾）/ `all`
5. 选择「绑定AI」：
   - `全部/不绑定`：不按业务线过滤
   - 指定 AI：按 AI 业务线过滤（来自 AI 配置中心的已配置模型）
6. 输入「关键词」：在“用户内容/机器人回复”中模糊匹配
7. 设置「每页数量」：控制列表加载条数

预期结果：下方表格展示符合条件的记录列表。

### 3.2 批量标注与绑定

1. 在列表中勾选需要操作的记录（列：选择）
2. 在「设置可学习/设置垃圾/绑定到AI/标签」中选择要应用的变更
3. 点击「✅ 应用到已选择记录」

预期结果：选中记录的标记/绑定/标签被写入，页面刷新后可按筛选条件复查。

### 3.3 删除垃圾数据

1. 勾选要删除的记录
2. 勾选「确认删除」
3. 点击「🗑️ 删除已选择记录」

预期结果：选中记录从学习数据集中删除。

## 4. 导出学习集（JSONL）

入口：进入 AI 学习中心后，选择「📦 导出学习集」标签页。

1. 选择「平台」：`telegram` / `whatsapp` / `all`
2. 选择「绑定AI」：
   - `全部/不绑定`：导出通用学习集
   - 指定 AI：导出该业务线学习集
3. 设置「导出数量上限」
4. 点击「📦 导出 JSONL」

预期结果：系统在当前租户目录生成导出文件并提示路径。

导出文件路径（按租户隔离）：
- `data/tenants/<tenant_id>/learning_exports/learnset-YYYYMMDD-HHMMSS.jsonl`

## 5. 安全与合规

- 学习数据包含用户内容与机器人回复，属于敏感数据；建议限制访问权限并避免外发。
- 导出文件建议按最小必要原则留存，交付给外部前需脱敏。
