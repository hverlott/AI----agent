# 系统设置与权限管理

Version: 2.5.0
Last Updated: 2026-01-12

## 1. 功能概述
本模块涵盖系统的基础设施配置（环境变量、密钥）、API 网关管理以及安全权限控制。

## 2. 操作步骤

### 2.1 系统配置自动化 (System Config)
侧边栏选择 **平台** -> **系统配置自动化**。

- **环境配置 (.env)**:
    - 查看当前系统环境变量状态。
    - 点击 **一键生成/更新** 可自动根据模板修复缺失配置。
- **敏感信息查看**:
    - 系统默认隐藏 API Key 等敏感信息。
    - 点击 **生成二次验证码** -> 终端查看验证码 -> 输入验证码 -> 点击 **查看明文** 解锁显示。

### 2.2 API 网关管理 (Gateway)
侧边栏选择 **平台** -> **API接口管理中心**。

- **路由管理**: 配置系统对外暴露的 API 路径 (e.g., `/audit`, `/reply`)。
- **流控策略**: 设置接口的限流阈值 (Rate Limit)，防止滥用。
- **鉴权配置**: 启用/禁用接口的 API Key 验证。

### 2.3 权限管理 (RBAC)
当前版本采用“系统账号登录 + 角色 + 租户隔离”的权限模型：

- **系统登录**：进入后台需要输入账号密码登录。
- **角色**：
  - SuperAdmin：可访问系统管理、API 网关等高权限模块，并可切换租户。
  - BusinessAdmin：可访问业务侧模块（AI 配置/编排/学习中心/技能中心等），租户固定绑定不可切换。
- **租户隔离**：配置与数据按租户隔离加载与展示。
