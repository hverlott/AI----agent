# 系统登录与快速入门

Version: 1.0
Last Updated: 2026-01-09

## 1. 系统启动与登录

本系统为本地化部署的 AI 客服管理工具，支持通过 Web 界面进行统一管理。

### 1.1 启动管理后台
1. 打开系统根目录 `D:\AI Talk`。
2. 双击运行 `start_admin.bat` (Windows) 或 `start_admin.sh` (Linux/Mac)。
3. 系统将自动打开默认浏览器，访问 `http://localhost:8501`。

![系统启动界面示意图]

### 1.2 身份与租户切换
系统支持多租户与多角色管理，适用于不同业务线的隔离管理。

1. **系统登录 (Login)**:
   - 进入后台后会出现“系统登录”表单。
   - 输入账号与密码后点击“登录”。
   - 若启用了 IP 白名单且当前 IP 不在白名单中，将直接拒绝访问。

2. **角色与权限 (Role)**:

| 角色 | 典型账号 | 租户切换 | 可见模块 |
| :--- | :--- | :--- | :--- |
| 超级管理员（SuperAdmin） | 由系统创建 | 可编辑侧边栏“租户ID” | 全部模块（含 API 网关/系统管理） |
| 业务管理员（BusinessAdmin） | 绑定到指定租户的业务账号 | 不可切换（固定绑定） | 业务侧模块（含 AI 配置/编排/学习中心/技能中心等） |

3. **租户切换 (Tenant)**:
   - 入口：左侧侧边栏「租户ID」。
   - SuperAdmin：可直接输入租户 ID（默认 `default`）。
   - BusinessAdmin：租户 ID 为只读，来源于账号绑定。
   - 切换后，配置与数据按租户隔离加载（例如 AI 配置、技能、学习数据、指标等）。

已废弃：旧版文档中的“租户下拉框”“默认无强制登录鉴权”“手动角色切换入口”，在当前版本已不适用，请以上述新路径为准。

## 2. 界面概览

系统主要分为以下几个功能区域：

- **侧边栏 (Sidebar)**: 全局导航、身份/租户切换、系统信息。
- **顶部状态栏**: 显示当前选中平台及其运行状态。
- **主工作区**: 根据选择的模块显示具体配置与数据。

## 3. 常见问题 (FAQ)

*   **Q: 启动后浏览器未自动打开怎么办？**
    *   A: 请手动复制终端显示的 URL (通常是 `http://localhost:8501`) 到浏览器访问。
*   **Q: 忘记了 Admin 密码？**
    *   A: 请使用系统管理员在「🛠️ 系统管理 > 租户与系统账号」创建/重置账号密码；业务管理员必须绑定租户。
*   **Q: 登录后提示“当前IP未在白名单中，禁止访问”怎么办？**
    *   A: 使用 SuperAdmin 账号进入「🛠️ 系统管理 > IP白名单」将当前 IP 加入白名单后重试。
*   **Q: 为什么切换租户后数据是空的？**
    *   A: 新租户默认没有任何配置，需重新进行初始化配置（账号录入、AI 配置等）。
