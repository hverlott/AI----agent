# Skills 技能中心

Version: 2.5.0
Last Updated: 2026-01-12

## 1. 功能用途

Skills 技能中心用于配置“可选增强能力”，按业务线（AI 业务线）对回复进行规则/Prompt 注入。

兼容性说明：Skills 为可选增强，默认行为不受影响；开启技能仅在匹配的回复路径中生效。

## 2. 入口与权限

- 面包屑：左侧菜单「🤖 AI 配置中心」>「🧩 技能中心」

| 角色 | 是否可用 | 租户范围 |
| :--- | :--- | :--- |
| 超级管理员（SuperAdmin） | 可用 | 可切换租户后管理不同租户技能 |
| 业务管理员（BusinessAdmin） | 可用 | 仅可管理绑定租户技能 |

## 3. 技能列表

入口：进入技能中心后，选择「📋 技能列表」。

1. 查看技能字段：ID、名称、启用状态、绑定 AI 业务线、更新时间、说明
2. 删除技能：
   - 选择「技能ID」
   - 勾选「确认删除」
   - 点击「🗑️ 删除技能」

预期结果：技能从当前租户中移除。

## 4. 新增/编辑技能

入口：进入技能中心后，选择「➕ 新增/编辑」。

### 4.1 选择编辑对象

- 新建技能：保持「编辑已有技能(可选)」为 `-`
- 编辑技能：在下拉框选择要编辑的技能 ID

### 4.2 配置字段

1. 输入「技能名称」「技能说明」
2. 设置「启用」开关
3. 选择「绑定到AI业务线」
   - `全部/不绑定`：通用技能
   - 指定 AI：仅对该业务线生效
4. 选择「技能类型」
   - `prompt`：以 Prompt/模板方式注入
   - `rule`：以规则说明方式注入（呈现方式一致，便于管理）
5. 选择「适用回复路径」
   - `kb_only`：仅知识库直答路径生效
   - `script_only`：仅剧本/SOP（编排）路径生效
   - `both`：两条路径都生效
6. 编辑「Prompt模板/规则说明」
7. 点击「💾 保存技能」

预期结果：技能写入当前租户，并在启用后参与运行时注入。

## 5. 生效规则（与回复路径的关系）

系统存在两类主要回复路径：

- 知识库直答（KB_ONLY）：启用技能中 `kb_only` 与 `both` 会被注入。
- 剧本/SOP（编排）：启用技能中 `script_only` 与 `both` 会被注入。

当技能绑定了 AI 业务线时：
- 仅在当前会话/路由实际选择的 AI 业务线匹配时生效；未绑定的通用技能可在多个业务线共用。

## 6. 安全与规范

- 不要在技能模板中写入 API Key、Token、密码等敏感信息。
- 技能内容会影响最终回复风格与合规性；建议先在测试环境验证，再在生产租户启用。
