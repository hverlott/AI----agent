# 🌐 多平台社交媒体管理系统

## 📋 系统概述

这是一个统一的社交媒体 AI 自动回复管理系统，支持多个主流社交平台。

### 支持的平台

| 平台 | 状态 | 功能 |
|------|------|------|
| 📱 **Telegram** | ✅ 已完成 | 私聊、群聊、频道、群发 |
| 💬 **WhatsApp** | 🚧 开发中 | 私聊、群聊 |
| 📘 **Facebook** | 📅 规划中 | Messenger + 主页 |
| 💙 **Messenger** | 📅 规划中 | 独立客户端 |
| 💚 **微信 WeChat** | 📅 规划中 | 个人号、公众号 |
| 📷 **Instagram** | 📅 规划中 | DM 自动回复 |
| 🐦 **Twitter/X** | 📅 规划中 | DM + 提及回复 |
| 💜 **Discord** | 📅 规划中 | 服务器 Bot |

---

## 🚀 快速开始

### 启动多平台管理中心

#### Windows
```cmd
start_multi_admin.bat
```

#### Linux/Mac
```bash
chmod +x start_multi_admin.sh
./start_multi_admin.sh
```

浏览器会自动打开 `http://localhost:8501`

## 🛠️ 技术栈

### 后端
- **Python 3.8+**
  - Telethon (Telegram)
  - OpenAI API
  - Streamlit (Web UI)

- **Node.js** (WhatsApp)
  - whatsapp-web.js
  - Express (可选)

### 前端
- **Streamlit**
  - 响应式设计
  - 实时更新
  - 多Tab 支持

### 数据存储
- **配置文件**
  - JSON (平台配置)
  - TXT (话术配置)
  - ENV (敏感信息)

## ⚠️ 重要提示

### 风险说明

| 平台 | 风险等级 | 说明 |
|------|---------|------|
| Telegram | 🟢 低 | 官方 API，安全可靠 |
| WhatsApp | 🟠 中 | 第三方库，有封号风险 |
| Facebook | 🟡 中低 | 官方 API，需审核 |
| 微信 | 🔴 高 | 严格限制，高风险 |

### 使用建议

1. **生产环境**
   - 优先使用官方 API
   - 申请企业认证
   - 遵守平台规则

2. **测试环境**
   - 使用备用账号
   - 小批量测试
   - 监控异常

3. **备份策略**
   - 定期备份配置
   - 保存重要对话
   - 多账号备份
